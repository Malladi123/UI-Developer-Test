Question 1 : Diff between this and let.

Let :

"let" is used for declaring a variables.
It has block scope just like const, that is it can be accessacble inside the block it has been declared only.
Means , when we declare a number with let within a for loop then that number has scope only within the loop. Tt becomes undefiened or throws exception when called before declaration.

this :

The "this" keyword refers to the current instance of the class. Here, the parameter name and the name of the class’s field are the same. Hence to avoid ambiguity, the class’s field is prefixed with the this keyword.
